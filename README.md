## Technologies used:

- `React.js`, `TypeScript` & `Chakra UI`.
- `Vite.js` as module bundler.

User is able to:

* Filter games by genre
* Filter games by platform
* Sort games
* Search for a game in the search field
* Toggle light/dark mode
